#tcp-port-scanner
This is a simple TCP port scanner written in Python. It allows scanning a single port or a range of ports on a given IPv4 host and reports whether ports are open or closed.

#Technologies
Python
Socket (standard library)

#Features
Validates IPv4 addresses and port ranges
Scans a single TCP port
Scans a range of ports
Displays open and closed ports
Simple command-line interface (CLI)

#How to run
Run the program:
python port_scanner.py

Follow the on-screen menu to select scanning mode and provide input values.

#Notes
The tool works only with IPv4 addresses
Scanning speed depends on network latency and timeout settings
This project is intended for educational purposes only
Do not scan hosts without proper authorization
